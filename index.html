Configuration Système

Paramètres MacBook Pro 2015

  • OS: macOS 12.7.6
  • RAM: 8 Go
  • CPU: Intel (sans GPU dédié)
  • Recommandations: Modèles quantifiés (GGUF, int4/int8)
  • Taille modèle optimal: 2-4B paramètres

Ports et Services

  • Ollama API: http://localhost:11434
  • Flask Backend: http://localhost:5000
  • Jupyter Lab: http://localhost:8888
  • ChromaDB: http://localhost:8000

Instructions de Déploiement

Installation Rapide

# 1. Cloner le repository
git clone https://github.com/avantgardepreparations-ops/mosaic-agent.git
cd mosaic-agent

# 2. Installer les dépendances Python
pip install -r requirements.txt

# 3. Lancer le serveur backend
python app.py

# 4. Ouvrir index.html dans votre navigateur

Prérequis Système

  • Python 3.8+
  • Docker (optionnel)
  • Homebrew (macOS)
  • 8 Go RAM minimum

Installation Outils IA

  • Ollama: brew install ollama
  • Whisper: pip install openai-whisper
  • ChromaDB: pip install chromadb
  • LangChain: pip install langchain
